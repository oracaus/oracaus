# Code of conduct

This project follows the
[Contributor Covenant, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
The full text is canonical at the link above; participants in this project
agree to abide by it.

## Scope

This code applies to all project spaces — the repository (issues, pull
requests, code review), the npm package, and any direct contact about the
project.

## Reporting

Reach out privately via
[LinkedIn](https://www.linkedin.com/in/przemyslawkalka/?locale=en-US).
Reports are handled in confidence; outcomes follow the enforcement ladder
described in the Covenant.
